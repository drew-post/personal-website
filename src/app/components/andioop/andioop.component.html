Discord Bot

Find the Github Repository for this project: here.

I started this Discord bot in 2019 to generate fun content in Discord servers I was in with my friends. The bot utilizes Discord,js, Javascript, Node.js to respond to corresponding commands following the astrick character, call APIs to generate content that is then formatted and sent back to the user, and send randomized links and images that are stored in the repo.

Some examples of the bot's commands include: sending an enlarged JPG of a user's avatar, generating a random dog or cat fact, generating a random joke, sending a link to an old vine, querying and sending back lyrics to requested songs, sending a random cowboy or clown picture, and sending my local time zone's time at the time requested. Some of these commands can be seen in the images provided.

Improvements I'd like to make to this project are to potentially invest in hosting the bot on a server, continue to build out new commands as well as expand upon existing commands to further optimize performance.
