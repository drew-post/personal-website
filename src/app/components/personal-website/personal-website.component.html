My Personal

Website

Welcome to my personal website porfolio! I started this project in 2022 as a way to explore more creative ways to use Angular, HTML, and CSS. I'm using Angular specifically to implement routes and lazy loading, I'm using Canva for some of the graphics, such as the homepage, and I'm using Github pages to host the website on Github. I've gone through a few iterations in regard to design for the website, ultimately trying to balance a clean and user friendly interface with fun accents, styles, and animations where appropriate. In addition to design, I'm trying to make the website as responsive as possible, so functionality and design remain consistent across all devices. By managing this project, I'm hoping to continue to learn and iterate on coding best practices as well as experiment with design in a contained way.

Some next steps and improvements I want to make include building out new and existing pages, iterating on website design, further optimizing the way the website handles comoponents/component structure, and continuing to come up with new, creative ideas for content on the website.

Thank you for visiting!
